# Imágenes para Viaje Mundo

Este directorio contiene todas las imágenes utilizadas en el sitio web de Viaje Mundo.

## Estructura de archivos de imagen

### Imágenes principales:
- `paris-card.jpg` - Imagen de París para la tarjeta de destino (400x250px)
- `tokyo-card.jpg` - Imagen de Tokio para la tarjeta de destino (400x250px)
- `paris-hero.jpg` - Imagen principal de París (1200x400px)
- `tokyo-hero.jpg` - Imagen principal de Tokio (1200x400px)
- `paris-main.jpg` - Imagen destacada de París para la página de categoría
- `tokyo-main.jpg` - Imagen destacada de Tokio para la página de categoría

### Imágenes de galerías:
- `paris-louvre.jpg` - Museo del Louvre (300x200px)
- `paris-montmartre.jpg` - Montmartre y Sacré-Cœur (300x200px)

## Formatos soportados
- JPEG para fotografías
- PNG para imágenes con transparencia
- GIF para animaciones (si las hay)

## Optimización
Todas las imágenes deben estar optimizadas para web:
- Fotografías: JPEG con 80% de calidad
- Logos/iconos: PNG optimizado
- Peso máximo recomendado: 150KB por imagen

## Derechos de autor
Todas las imágenes utilizadas deben tener licencia libre o estar bajo Creative Commons.
Ver el archivo links.html para las atribuciones completas.

## Nota para desarrollo
En un entorno de producción real, estas imágenes serían reemplazadas por:
- Fotografías profesionales de alta calidad
- Imágenes con licencias apropiadas
- Contenido optimizado para diferentes dispositivos
- Formatos modernos como WebP donde sea compatible

Para fines educativos de esta práctica, se pueden usar placeholders o imágenes libres de derechos.
